🏎️ Formula 1 Lap Time & Race Performance Analysis
Barcelona-Catalunya Grand Prix – MATLAB Data Project
📌 Overview

This project analyses Formula 1 race performance data from the Barcelona-Catalunya Grand Prix using MATLAB. The objective is to process lap time and classification data to identify performance trends across drivers and teams, with a focus on pace, consistency, and race progression.

The project is part of my self-directed learning pathway toward a career in motorsport engineering and data analysis, with a focus on developing skills relevant to race engineering environments.

🎯 Project Objectives
Import and structure race data using MATLAB
Analyse lap time trends across multiple drivers
Compare race pace and consistency between competitors
Evaluate position changes from grid to finish
Identify performance patterns across teams
Develop clear and professional motorsport-style visualisations
🛠️ Tools & Skills Used
MATLAB (primary analysis tool)
Data structures (tables, arrays)
CSV data import and handling (readtable)
Data visualisation (plot, bar, legend, grid on)
Basic statistical analysis (mean, standard deviation)
Motorsport performance interpretation
📊 Data Handling

The dataset used in this project includes:

Driver names
Team affiliations
Lap-by-lap performance data (simulated structure for development stage)
Race classification (finish position / DNF / NC)
Laps completed
Grid vs finish position comparison

Data is structured in MATLAB using tables for efficient analysis and filtering.

📈 Key Analyses Performed
1. Lap Time Analysis
Visual comparison of lap-by-lap performance between drivers
Identification of pace trends across the race distance
2. Race Classification Table
Full driver list with team affiliation
Final race status (P1–P20, DNF, NC)
Laps completed per driver
Position changes from grid to finish
3. Position Gain Analysis
Calculation of positions gained/lost during the race
Identification of strongest race progressions
4. Team Performance Overview
Comparison of finishing reliability across teams
Basic evaluation of race completion rates
📊 Key Insights
Small differences in average lap time can translate into significant performance gaps over a race distance
Race consistency plays a major role in overall finishing position
Position change analysis highlights racecraft and strategic advantage beyond qualifying performance
Reliability (DNFs vs finishers) significantly impacts team outcomes

These insights demonstrate the importance of combining raw pace with consistency and race execution — a key principle in modern motorsport engineering.
